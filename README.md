# COMP0118 Coursework 2 — MRI T2 Relaxometry

Analysis of MRI T2 relaxometry data using MATLAB.

## Structure
- `nifti_tools/` — NIfTI file reading toolbox (Jimmy Shen).
- `my_scripts/` — coursework analysis scripts.
- `T2_matlab_material/` — lecture material.
- `data/` — control data.
- `data-adolescent` — our main dataset of interest. You need to download it from the onedrive and place it here.
- `metadata` — additional information about the dataset.

## Setup
```matlab
addpath('./nifti_tools/')
```

For python setup instructions look at `notes.md` in the `python` directory.
